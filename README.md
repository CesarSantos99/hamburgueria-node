# hamburgueria-node

## Tecnologias:

*Node.js<br>
*express <br>
*Uuid<br>
*cors

## Sobre:

Apis do tipo get para fazer as comandas, post para o pedido, put para deixar os clientes atualizados, e delete para deletar as comandas
